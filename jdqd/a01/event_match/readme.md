## 使用前准备

    将对应的模型存放到对应的文件夹下，然后运行main.py文件即可
    
## algor
    
    存放模型测试与预测所有的代码
    
    ### common

        - utils.py 训练模型以及预测时构建数据生成器代码    

    ### predict
    
        - execute.py 预测过程中加载事件列表、加载模型、预测以及预测结果的整理代码
        
        - get_abstract.py 获取摘要的代码
            
    ### train

        - match_model_train.py 匹配模型训练代码
        
## config

    存放模块参数类
